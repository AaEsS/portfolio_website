some Projects

WorldTime [Dart, Flutter] is a mobile app that uses web scraping and API calling to retrieve and show time data of regions from worldtimeapi.org. The app starts by automatically fetching the current time of the user's region. The user can choose from all of the regions presented in a scrollable list, or use an implemented search bar to find their desired one.

DotDodge [C#, Unity] is a short game where the player tries to beat it by dodging all homing bullets/missiles. The game gets harder the longer you survive. The player wins when the timer hits zero.

Game published here

Save My Face [C#, Unity] is a personal commercial mobile game about my face controlled by the player. The goal is to "save my face" from the bullets that are shot by a cannon constantly facing the player. There are powerups that can help the player survive longer. At the end, a score of how many bullets survived is shown. A leaderboard and ads are implemented as well.

Game published here

Game Without Art [C#, Unity] is similar to Save My Face. This one has more powerups and the ability to use more than one at once. Instead of a cannon shooting, each bullet grows and pops to three small ones that act like their parent.
